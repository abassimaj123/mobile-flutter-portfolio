SUBJECT: FOIA Request – San Francisco Parking Meter and Street Parking Data

This is a request under the California Public Records Act (CPRA) for the following public records:

1. PARKING METER INVENTORY & RATES
   Complete dataset of all SFMTA parking meters and regulations:
   - Meter ID / sensor ID
   - Street location (address)
   - Block number, lot number
   - Latitude and longitude coordinates
   - Rate schedule (price per hour, e.g., $0.75/hr)
   - Operating hours (e.g., 8 AM - 10 PM, which days)
   - Maximum stay (in minutes or hours)
   - Zone / district name
   - Effective date of current rates

   Requested format: CSV, Excel (.xlsx), JSON, or GIS shapefile

2. STREET PARKING REGULATIONS (Curb Space Rules)
   Complete street parking regulation database by street segment:
   - Street name
   - From cross street / intersection
   - To cross street / intersection
   - Side of street
   - Regulation type (NO PARKING, LOADING ZONE, METER, etc.)
   - Detailed rule (e.g., "No Parking 7 AM - 9 AM, Mon-Fri")
   - Max stay duration
   - Permit requirements (if applicable)
   - Effective date

   Requested format: CSV, Excel, JSON, or GIS-compatible format

3. PARKING SWEEPING / STREET CLEANING SCHEDULE
   Complete street sweeping schedule:
   - Street name (or segment)
   - Sweeping day(s) of week (e.g., Monday, Thursday)
   - Sweeping hours (e.g., 8 AM - 10 AM)
   - Zone / area designation
   - Current effective schedule

   Requested format: CSV, Excel, JSON, or text table

4. OPTIONAL: PARKING LOT INVENTORY (Public Parking)
   If available, locations of public parking lots operated by SFMTA:
   - Lot address
   - Capacity (number of spaces)
   - Rate (if any)
   - Hours of operation
   - Contact information

5. OPTIONAL: API DOCUMENTATION
   If SFMTA maintains a public parking data API, please provide:
   - API endpoint documentation
   - Authentication requirements
   - Data format and schema
   - Rate limits (if applicable)

JUSTIFICATION:
This data will be used to develop a mobile application ("ParkSmart") that provides San Francisco residents and visitors with accurate, accessible parking rule information. The app will reduce parking violations, improve driver behavior, and provide a public service at no cost to SFMTA.

WAIVER OF FEES:
I request a waiver of search and copying fees, as this request is for public information that will be used for public benefit (educational/informational mobile app, provided free).
